Search for emoji and copy them to your clipboard.

![emoji-cli](./assets/emoji-cli.gif)

## Install

Grab a binary from the releases page or build with cabal:
```
git clone https://github.com/aloussase/emoji-cli
cd emoji-cli
cabal install
```
